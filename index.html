OOP - Encapsulation

  • bundling of data and methods operating on this data into one unit
  • hide the internal representation
  • control access to the internal state

Access Modifiers

  • private - same class only
  • no modifier - same package only
  • protected - same package and subclasses
  • public - all classes

What is Spring

  • framework
  • easily Build Java Enterprise Applications


Core Functionalities

  • dependency injection mechanism (IoC)
  • configuration model
supports a wide range of application scenarios:
web, loaders, embedded, cloud

Core Concept

Inversion of Control

Defining dependencies only through constructor args, factory methods, set after constructed

Inverse of object defining dependencies directly.


// defining dependency directly
public class DirectDep{
  DepA depA = new MyDepA();
  DepB debB = new MyDepB();

  public DirectDep(){
    ...
  }
}
            

// IoC through Spring
public class DirectDep{
  DepA depA;
  DepB debB;

  @Autowired
  public DirectDep(
    DepA depA, DepB depB){
    this.depA = depA;
    this.depB = depB;
  }
}
